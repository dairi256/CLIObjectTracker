# CLIObjectTracker - A Lightweight Application to fetch and display orbital data, enhanced with ML
CLIObjectTracker is a C# Console Application, leveraging the use of Celestrak to fetch TLE data for a wide range of entities, including but not limited to:
- Satellites
- Spacecrafts (Voyager, etc)
- Other related objects that may be tracked by satellites

## Features
- Fetches TLE (Two-Line Element) Data from Celestrak
- A simple and clean CLI, suited well for even beginners
- Modular project structure for future expansion/feature implementation

## Planned Changes/Features
- Orbit Propagation using SGP4
- ML Algorithm implementation to predict orbital drifts
- Command Parsing (e.g., 'Track ISS'), creating a simpler use for everyone
- Exporting prediction results of ML algorithm
